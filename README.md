# Emissa Premium MVP Demo

A single-page premium demo MVP for the Emissa Compliance Operating System.

## What is included

- Premium dark SaaS command center UI
- Fully working sidebar navigation
- Overview dashboard with animated network globe
- Six-product suite cards
- Emissa Network buyer portal view
- Regulatory Intelligence Center
- Reports & Audit Center
- Five-year build roadmap
- Individual module screens for Core, EPR, CBAM, Climate Risk, and LCA Studio
- Working buttons, modals, simulated export, sync actions, global search, and AI assistant

## How to preview

Open `index.html` directly in a browser or deploy the repository through GitHub Pages, Vercel, Netlify, or any static host.

## Demo flow

1. Start at Overview.
2. Click `Start 4-Min Demo`.
3. Show the ARR curve, compliance score, and active buyer obligations.
4. Open Product Suite.
5. Click each module card.
6. Open Network to show the buyer/supplier moat.
7. Open Regulatory Center to show mandatory demand.
8. Open Reports to show buyer-ready outputs.
9. Finish with Roadmap.
